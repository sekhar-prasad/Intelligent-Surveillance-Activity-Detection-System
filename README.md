# Intelligent Surveillance & Activity Detection System

## Overview
An AI-based surveillance system that detects objects and analyzes activities in real-time using OpenCV and MobileNet-SSD.

## Features
- Real-time object detection (Person, Car, etc.)
- Activity detection using motion analysis
- Restricted zone monitoring
- Intrusion alert system

## Tech Stack
- Python
- OpenCV (DNN Module)
- MobileNet-SSD
- NumPy
